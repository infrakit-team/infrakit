# Introduction

Welcome to InfraKit! InfraKit is a lightweight, self-hostable dashboard for your application's infrastructure components. It allows you to monitor, manage, and interact with various modules, starting with a powerful in-memory Key-Value store.

## What is InfraKit?

InfraKit provides:
- A central SDK to integrate different infrastructure modules.
- A pre-built UI dashboard for at-a-glance observability.
- Adapters for popular web frameworks like Express and Hono.
- An extensible module system, starting with a Key-Value store.
